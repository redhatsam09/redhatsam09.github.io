Install Nift on Your Platform

Pick the method that fits your workflow. Whether you prefer package managers, binary releases, or building from source, this guide walks you through getting Nift running quickly.

Download Release Binaries

Linux

Choose between Snap, Flatpak, or compiling from source. All approaches deliver the same fast CLI tooling.

Snapcraft (recommended for Ubuntu)
  1. Install Snap if needed:
    sudo apt-get install snapd
  2. Install Nift:
    sudo snap install nift --classic
  3. Remove Nift when finished:
    sudo snap remove nift
Flatpak
  1. Install Flatpak and add Flathub:
    flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
  2. Install Nift for your user:
    flatpak install --user https://flathub.org/repo/appstream/cc.nift.nsm.flatpakref
  3. Optional: add shell aliases to ~/.bashrc:
    alias nift="flatpak run cc.nift.nsm"
    alias nsm="flatpak run cc.nift.nsm"
Build from Source
  1. Clone the repository:
    git clone https://github.com/nifty-site-manager/nsm.git
  2. Install build dependencies (g++, make, LuaJIT or system Lua):
    sudo apt-get install build-essential git luajit
  3. Build and install:
    make -j$(nproc)
    sudo make install

FreeBSD

Use the official port or fetch binaries through pkgng.

Install from Ports 
cd /usr/ports/www/nift
sudo make install clean
Install via pkgng 
sudo pkg install www/nift
Uninstall 
# Using ports
doas make -C /usr/ports/www/nift deinstall

# Using pkgng
doas pkg delete www/nift

Gentoo

Add the overlay and pick the stable or live ebuild.

  1. Add overlay:
    sudo layman -f -o https://raw.githubusercontent.com/NuLL3rr0r/gentoo-overlay/master/repositories.xml -a NuLL3rr0r
  2. Install stable:
    sudo emerge -atuv www-apps/nift
  3. Install live (development):
    sudo emerge -atuv '=www-apps/nift-9999'
  4. Pin live build:
    echo '=www-apps/nift-9999 **' | sudo tee /etc/portage/package.accept_keywords/www-apps_nift
  5. Remove:
    sudo emerge -C www-apps/nift

macOS

Install with Homebrew or compile from source.

Homebrew 
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install nift
From Source 
git clone https://github.com/nifty-site-manager/nsm.git
cd nsm
make -j$(sysctl -n hw.ncpu)
make install PREFIX=/usr/local

Windows

Grab a Chocolatey package or extract the executable release bundle.

Chocolatey 
choco install nift
Executable Bundle
  1. Download Windows.zip from the releases page.
  2. Extract lua51.dll, nift.exe, and nsm.exe.
  3. Copy the files into a directory on your %PATH% (for example C:\Windows\System32).
Build from Source

Install a C++ toolchain (MSYS2 or Visual Studio), clone the repo, then run make mingw from the provided makefile to produce nsm.exe and nift.exe.
